課程筆記：API 與 Fetch 基礎

1. 什麼是 API？
API (Application Programming Interface) 就像是餐廳的服務生，你（前端）把點單交給服務生（API），服務生去廚房（資料庫）拿菜，再把菜（資料）端回給你。

2. HTTP Request (GET) 與 Fetch
在 JavaScript 中，我們使用 fetch() 函數來向 API 請求資料。最常見的方法是 GET（拿取資料）。

3. JSON 格式
JavaScript Object Notation (JSON) 是一種輕量級的資料交換格式。它看起來像 JS 的物件，是目前 Web API 傳遞資料的絕對主流。
